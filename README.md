# mbti_personality_classifier
!В папке "СТРУКТУРИРОВАННЫЙ ПРОЕКТ" находится структурированый проект с полностью готовыми кодами. Изначально мы загрузили файлы, которые у нас были и чуть позже занялись структурой проекта!


Мы сделали проект, который умеет анализировать текст человека о себе и определять его тип личности по системе MBTI (всего бывает 16 типов). Также мы сделали готовую модель в виде Telegram-бота.


## Как устроена наша модель
* **Разбили задачу на 4 части:** Вместо того чтобы учить модель угадывать сразу один из 16 типов, мы создали **4 независимые модели** — по одной на каждую шкалу MBTI (`I/E`, `N/S`, `T/F`, `J/P`).
* **Алгоритм:** Использовали `RandomForestClassifier` из 100 деревьев решений и текстовый векторайзер `TF-IDF`. 
* **Понимает русский язык:** Мы настроили очистку текста так, чтобы она убирала ссылки и мусор, но при этом отлично понимала и русскую кириллицу, и английский текст.
* **Защита от коротких текстов:** Бот принимает сообщения строго от 50 слов. Если написать меньше, он попросит добавить текста, потому что по двум словам невозможно точно определить психологические маркеры.

## Структура папок
* `plots/` — график распределения классов в нашем датасете.
* `src/` — папка со всем нашим кодом:
  * `create_models.py` — здесь мы чистим данные, настраиваем русский язык и обучаем наши 4 модели.
  * `predict.py` — скрипт для быстрой проверки предсказаний через консоль.
  * `bot.py` — код нашего Telegram-бота для защиты проекта.
* `.env` — скрытый файл, где лежит токен от бота.
* `requirements.txt` — список библиотек, которые нужны для работы.

## Как запустить наш проект в VS Code

### 1. Открываем проект
1. Склонируйте репозиторий или просто откройте папку проекта через VS Code.
2. Открываем встроенный терминал в VS Code.

### 2. Настраиваем окружение
В терминале VS Code создаем и включаем виртуальное окружение:

```bash
# 1. Создаем окружение:
python -m venv venv

# 2. Активируем его:
# Windows:
venv\Scripts\activate
# Mac или Linux:
source venv/bin/activate

# 3. Устанавливаем все нужные библиотеки:
pip install -r requirements.txt
```

### 3. Добавляем токен бота
Создаём в корне проекта файл`.env` и вставьте туда токен бота, который выдал бот создающий Telegram-боты:
```env
BOT_TOKEN=8806764659:AAHrgVPfcBvxVUkpjof7UgMjGTzfEKpfrws
```

### 4. Обучаем модели
Чтобы обучить наш Случайный лес и сохранить файлы моделей в папку `models`, запускаем в терминале:
```bash

```

### 5. Запуск Telegram-бота
Теперь запускаем самого бота для проверки:
```bash

```
